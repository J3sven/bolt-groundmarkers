Instance Layout System Updated

The way instance layouts are stored has been improved to support large instances (e.g., Elite Dungeons). Unfortunately, this means existing instance layouts are incompatible with this version.

  • Non-instance world layouts are NOT affected.
  • Your existing instance layouts need to be recreated.
  • You can delete old instance layouts now or manually later
